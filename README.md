FreshMart

FreshMart is a modern, responsive grocery e-commerce web application built with React, Vite, TypeScript, and Tailwind CSS.
This project serves as the base for developing a full-featured online grocery store where users can browse products, interact with dynamic UI components, and eventually integrate backend services for cart management, user accounts, and checkout flows.
